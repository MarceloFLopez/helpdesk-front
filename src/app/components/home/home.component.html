Help Desl Dicas de um DEV

Olá! Seja bem vindo ao Sistema Help Desk desenvolvido por mim no curso Desenvolvedor FullStak.
Esse é um sistema para gerênciamento de chamados simulando um simples Help Desk desenvolvido em Angular e Springboot3.
O Sistema está irá rodar na nuvem sendo o front-end na Vercel e o back-end Railway.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação via Token JWT,
Persistência de dados usando JPA integrando com o banco de dados MySQL.

Código fonte sera disponibilizado no github para fins academicos e colaborações.
